# Dawish Salla Theme

ثيم خاص لعطارة الدويش مبني على Salla Twilight.

## GitHub structure

Make sure these files are in the repository root:

- `twilight.json`
- `package.json`
- `src/`
- `README.md`

Do not upload a parent folder such as `dawish-salla-theme-starter/` as the only item in the repository.

## Local preview

```bash
npm install
salla login
salla theme preview --with-editor --browser=chrome
```


## تحديث إعدادات الثيم
هذه النسخة تضيف صفحة إعدادات موسعة داخل لوحة سلة، تشمل:
- خيارات الصور والقوائم.
- خيارات أعلى الصفحة والهيدر.
- خيارات أسفل الصفحة والفوتر.
- خيارات صفحة المنتج.
- خيارات متقدمة للبحث وشاشة التحميل.

ارفع محتويات هذا المجلد إلى جذر مستودع GitHub، ثم اختر فرع `main` من لوحة الثيم في سلة.
