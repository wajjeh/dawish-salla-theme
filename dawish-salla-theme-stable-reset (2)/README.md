# Dawish Salla Theme — Launch Ready

ثيم خاص لعطارة الدويش مبني على Salla Twilight، نسخة محسنة للإطلاق.

## البنية الصحيحة في GitHub

لازم تكون الملفات في جذر المستودع مباشرة:

- `twilight.json`
- `package.json`
- `src/`
- `README.md`
- `LAUNCH_CHECKLIST.md`

لا ترفع مجلد خارجي مثل `dawish-salla-theme-launch-ready/` داخل المستودع.

## طريقة الرفع السريعة

افتح هذا المجلد وارفع محتوياته فقط إلى GitHub:

```text
src
README.md
LAUNCH_CHECKLIST.md
package.json
twilight.json
```

## المعاينة المحلية

```bash
npm install
salla login
salla theme preview --with-editor --browser=chrome
```

## تحسينات نسخة الإطلاق

- واجهة رئيسية أقوى مع Hero، أقسام، منتجات، بنرات، قصة المتجر، ومميزات.
- شاشة تحميل احترافية بشعار المتجر من سلة تلقائيًا.
- هيدر محسن للجوال والكمبيوتر مع بحث كبير وحساب ومفضلة وسلة.
- أزرار عائمة: واتساب ورجوع للأعلى.
- تحسين صفحة المنتج مع بطاقة ثقة، إجراءات شراء واضحة، وزر شراء ثابت للجوال.
- تحسينات CSS للجوال، سرعة القراءة، ووضوح السعر والزر.
- إعدادات إضافية داخل لوحة سلة للتحكم بالإطلاق بدون تعديل الكود.

## تحديث Mobile Polish v2

تم إضافة طبقة تحسينات مخصصة للجوال وصفحات الشراء:

- إعادة تقديم خيارات الأحجام كاختيارات واضحة وبسيطة بدل ظهورها كمنتجات مصغرة.
- إخفاء صور خيارات الحجم داخل الاختيارات قدر الإمكان مع الحفاظ على مدخلات سلة الأصلية بدون كسر وظائفها.
- إضافة رسالة “تم اختيار” بعد اختيار الحجم وتحديثها أيضًا في زر الإضافة الثابت بالجوال.
- ضبط زر إضافة للسلة الثابت ليظهر فوق شريط التنقل السفلي بدون تداخل.
- رفع زر واتساب وسهم الرجوع للأعلى في صفحة المنتج حتى لا يغطون على زر الإضافة.
- تحسين شكل السلة وملخصات الدفع والكوبون قدر الإمكان من داخل الثيم.
- إضافة تحسينات عامة لصفحة إتمام الطلب، مع ملاحظة أن بعض عناصر Checkout تتحكم بها منصة سلة.

بعد الرفع على GitHub، اختبر تحديد الحجم في صفحة المنتج، ثم إضافة المنتج للسلة، ثم ظهور الحجم داخل السلة وإتمام الطلب.

## Homepage Visible Fix

This build makes the home page visible immediately by rendering the Dawish launch sections directly from `src/views/pages/index.twig`. It prevents the Salla preview from showing a blank home page before adding homepage blocks manually.
